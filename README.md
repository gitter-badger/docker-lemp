# docker-lemp

Run Nginx, php-fpm and MariaDB using [Docker]

## Requirementos
Install [Docker] and [Compose]

## Modo de uso
```
docker-compose up -d
docker-compose logs
docker-compose stop
docker-compose rm


[Docker]:                      https://www.docker.io/
[Compose]:                     http://docs.docker.com/compose/install/
